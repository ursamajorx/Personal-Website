What I Do

Involves creating a flexible and responsive layout that can adapt to different screen sizes and devices. I use various techniques such as fluid grids, flexible images, and media queries to ensure that the website is accessible and user-friendly across all devices. I ensure that the website looks great and functions well, regardless of the user's device.

Frontend Development
Involves creating the visual elements of a website that users interact with directly. I use web technologies such as HTML, CSS, and JavaScript to build user-facing components such as layouts, typography, color schemes, and user interfaces. Backend Development
Involves creating and managing the server-side components of a website. This includes the application logic, database management, and server configuration. I use various programming languages such as Javascript and Python to build and maintain the server-side components. My job is to ensure that the website is scalable, reliable, and can handle large amounts of data and traffic.
